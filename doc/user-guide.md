# User Guide

## 1. Getting Started

### Create an Account
- Go to the homepage and click **Register**
- Choose your role:
  - Student
  - Instructor
  - Teaching Assistant (TA)

### Log In
- Enter your credentials to access your dashboard

---

## 2. Dashboards

### Student Dashboard
- View enrolled courses
- Track extra credit
- Access glossary terms

### Instructor Dashboard
- Create and manage courses
- Add glossary terms by week or topic
- Set extra credit values

### TA Dashboard
- Review student submissions
- Compare entries
- Select best submissions

---

## 3. Course Enrollment (Students)

1. Go to **Courses**
2. Search or enter a course code
3. Click **Enroll**

---

## 4. Glossary System

Each course contains weekly or topic-based glossary terms defined by the instructor.

---

## 5. Submitting Definitions (Students)

### Steps
1. Open a course and select a glossary term
2. Click **Submit Definition**
3. Add:
   - Definition (required)
   - Optional:
     - Diagram or image
     - Example or related concept

### Rules
- You can submit 1–2 definitions per week
- Each term has a submission cap
- Once the cap is reached, submissions close

---

## 6. Viewing Submissions

- All submissions are visible to students
- You can compare multiple definitions
- Only one submission is officially approved & added to the glossary

---

## 7. Approval Process (TA / Instructor)

Submissions are evaluated based on:
- Accuracy
- Clarity
- Completeness
- Visual quality (if applicable)

### Results
- One submission is approved and added to the official glossary
- The selected student earns extra credit (typically 5 points)

---

## 8. Example Workflow

1. Enroll in a course  
2. Review glossary terms  
3. Select a term  
4. Submit a definition  
5. Compare with other submissions  
6. Get selected and earn extra credit   
7. Prepare for exams using compiled glossary
